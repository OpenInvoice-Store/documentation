## Product List View

The product list provides:

- **Search** - Search by name, SKU, or description
- **Filters** - Filter by category, price range, tax rate
- **Sorting** - Sort by name, price, category, or date added
- **Bulk Actions** - Select multiple products for batch operations

## Product Analytics

View product insights:

### Usage Statistics

- **Times Used** - How many invoices include this product
- **Total Revenue** - Revenue generated from this product
- **Average Price** - Average selling price
- **Last Used** - Date last added to an invoice

### Product Performance

- **Top Products** - Most frequently used products
- **Revenue by Product** - Products generating most revenue
- **Product Trends** - Usage over time

## Product Categories

### Creating Categories

1. Go to **Products**
2. Click **Categories** or **Manage Categories**
3. Click **New Category**
4. Enter category name
5. Add description (optional)
6. Save

### Assigning Products to Categories

1. Open the product
2. Select category from dropdown
3. Save

Or bulk assign:
1. Select multiple products
2. Click **Bulk Edit**
3. Select category
4. Apply

### Category Management

- **Edit Categories** - Update category names
- **Delete Categories** - Remove categories (products remain)
- **Reorder Categories** - Organize category display order

## Product Variants

Create product variants for:
- Different sizes
- Different colors
- Different configurations
- Different pricing tiers

### Creating Variants

1. Open the base product
2. Click **Add Variant**
3. Enter variant details:
   - Variant name
   - SKU (if different)
   - Price (if different)
4. Save

## Product Pricing Strategies

### Fixed Pricing

Set a fixed price that doesn't change:
- Standard products
- Services with set rates
- Subscription products

### Variable Pricing

Allow price adjustment per invoice:
- Custom quotes
- Negotiated pricing
- Volume discounts

### Price Tiers

Set up pricing tiers:
- Quantity-based discounts
- Customer-specific pricing
- Promotional pricing

## Product Inventory (Optional)

If inventory tracking is enabled:

### Track Stock

- **Stock Quantity** - Current inventory level
- **Low Stock Alert** - Set minimum threshold
- **Stock Updates** - Update when products are used

### Inventory Reports

- **Low Stock** - Products below threshold
- **Out of Stock** - Products with zero inventory
- **Stock Movements** - Inventory changes over time

## Product Images

### Image Management

- **Upload** - Add product images
- **Replace** - Update existing images
- **Delete** - Remove images
- **Reorder** - Set primary image

### Image Best Practices

- Use high-quality images
- Consistent sizing
- Optimize file size
- Add alt text for accessibility

## Product Bulk Operations

### Bulk Edit

Update multiple products at once:
1. Select products
2. Click **Bulk Edit**
3. Choose fields to update:
   - Category
   - Tax rate
   - Price (percentage or fixed)
4. Apply changes

### Bulk Delete

Remove multiple products:
1. Select products
2. Click **Delete**
3. Confirm deletion

### Bulk Export

Export selected products:
1. Select products
2. Click **Export**
3. Choose format
4. Download

## Product Templates

Create product templates for:
- Common service packages
- Product bundles
- Standard offerings

### Using Templates

1. Create a template product
2. When creating invoice, select template
3. Customize as needed
4. Save

## Product Search and Filtering

### Advanced Search

Search by:
- Product name
- SKU
- Description
- Category
- Price range

### Saved Filters

Save frequently used filters:
1. Set up filter criteria
2. Click **Save Filter**
3. Name the filter
4. Use saved filter anytime
