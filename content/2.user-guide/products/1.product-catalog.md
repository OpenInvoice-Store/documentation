## What is the Product Catalog?

The product catalog allows you to:
- Save products and services for quick reuse
- Maintain consistent pricing
- Set default tax rates
- Add product images
- Organize products by category

## Creating a Product

### Step 1: Navigate to Products

1. Go to **Products** in the main menu
2. Click **New Product**

### Step 2: Enter Product Details

Fill in the product information:

- **Name** - Product or service name (required)
- **Description** - Detailed description (optional)
- **SKU** - Stock keeping unit (optional)
- **Price** - Default price (required)
- **Tax Rate** - Default tax percentage (optional)
- **Category** - Product category (optional)
- **Image** - Product image (optional)

### Step 3: Save

Click **Save** to add the product to your catalog.

## Adding Products to Invoices

When creating an invoice:

1. Click **Add Item** or **Add Product**
2. Select **Product from Catalog**
3. Search or browse products
4. Select the product
5. Product details are auto-filled:
   - Description
   - Unit price
   - Tax rate
6. Adjust quantity and price if needed
7. Click **Add**

## Product Management

### Editing Products

1. Go to **Products**
2. Click on the product
3. Click **Edit**
4. Make changes
5. Click **Save**

> **Note**: Editing a product does not change existing invoices that use it.

### Deleting Products

1. Open the product
2. Click **Delete**
3. Confirm deletion

> **Warning**: Deleting a product does not remove it from existing invoices.

### Duplicating Products

Create similar products quickly:

1. Open the product
2. Click **Duplicate**
3. Edit as needed
4. Save as new product

## Product Categories

Organize products by category:

### Creating Categories

1. Go to **Products**
2. Click **Categories** (if available)
3. Create new category
4. Assign products to categories

### Filtering by Category

- Filter product list by category
- Find products quickly
- Organize catalog

## Product Images

Add images to products:

1. Open the product
2. Click **Upload Image**
3. Select image file
4. Crop/resize if needed
5. Save

Image guidelines:
- Supported formats: JPG, PNG, WebP
- Recommended size: 800x800px
- Max file size: 5MB

## Product Pricing

### Setting Prices

- **Default Price** - Set when creating product
- **Override Price** - Change price when adding to invoice
- **Currency** - Uses organization default currency

### Price Updates

When you update a product price:
- Existing invoices are not affected
- New invoices use updated price
- Can override per invoice

## Product Tax Rates

Set default tax rates for products:

- **Tax Rate** - Percentage (e.g., 8.5 for 8.5%)
- **Tax Exempt** - Mark product as tax-exempt
- **Override** - Can override when adding to invoice

## Best Practices

1. **Complete Information** - Fill in all product details
2. **Use Categories** - Organize products logically
3. **Add Images** - Visual products help customers
4. **Set Default Prices** - Save time when creating invoices
5. **Regular Updates** - Keep prices and descriptions current

## Import/Export Products

### Export Products

1. Go to **Products**
2. Click **Export**
3. Choose format (CSV, Excel)
4. Download

### Import Products

1. Go to **Products**
2. Click **Import**
3. Upload CSV or Excel file
4. Map columns
5. Import
